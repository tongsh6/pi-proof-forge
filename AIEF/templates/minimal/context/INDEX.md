# 上下文索引模板

这是上下文导航的模板起点。

## 建议结构

    context/
        business/     # 领域模型、业务规则（低频更新）
        tech/         # 架构、API、编码约定（中频更新）
        experience/   # 经验教训、最佳实践（高频更新）

## 快速导航示例

- 业务知识：context/business/DOMAIN.md
- 技术架构：context/tech/ARCHITECTURE.md
- 仓库快照：context/tech/REPO_SNAPSHOT.md
- 经验库：context/experience/INDEX.md
