# v0.1.4

## Summary
- 交付自动投递单平台（Liepin）MVP 脚手架。
- 增加可直接执行的 submission CLI 入口与 dry-run。

## Highlights
- 新增 `tools/submission/run_submission.py` 统一入口（模块调用：`python3 -m tools.submission.run_submission`）。
- 新增 `tools/submission/liepin.py` 平台适配层（MVP）。
- 支持参数校验与执行步骤预演（`--dry-run`）。

## Breaking Changes
- None

## Migration Guide
- 使用新命令进行预演：
- `python3 -m tools.submission.run_submission --platform liepin --job-url <url> --resume <pdf> --profile <yaml> --dry-run`
