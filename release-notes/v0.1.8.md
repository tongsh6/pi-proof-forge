# v0.1.8

## Summary
- 实现企业排除策略（company exclusion）全链路：DISCOVER 阶段过滤、GATE 阶段兜底、策略页与系统设置 UI、Sidecar 持久化与 API，与核心规范对齐。

## Highlights
- **策略层**：`tools/policy/exclusions.py` 支持 `exact:` / `contains:` 规则，`gate.py` 门禁校验、`audit.py` 审计；配置从 policy YAML 与 sidecar settings 加载。
- **CLI 集成**：`run_matching_scoring.py`、`run_pipeline.py` 在 discovery 与 gate 中应用排除规则，命中企业不进入投递候选。
- **Sidecar**：settings 新增 `exclusion_list`、`excluded_legal_entities` 读写，Policy 页与系统设置可编辑并持久化。
- **UI**：策略页展示门禁快照与排除企业/主体编辑；系统设置保留排除列表入口；中英文 i18n 与设计稿更新。
- **文档**：OpenSpec、AIEF 计划与 workflow 阶段文档同步企业例外清单语义与 DISCOVER/GATE 分工。

## Breaking Changes
- None. 企业排除为新增能力，既有 pipeline 与配置保持兼容。

## Migration Guide
- 若需启用企业排除：在策略页或系统设置中配置排除企业（展示名/别名）与排除主体（法人实体），保存后 sidecar 持久化，后续 run 自动生效。
- 规则格式：每行一条，支持前缀 `exact:`、`contains:`；无前缀按展示名精确匹配。
